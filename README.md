# API de Logística - Integración Envia.com

Este proyecto contiene una API RESTful para interactuar con los servicios de Envia.com.

## Descripción

La API proporciona endpoints para:
- Cotizar envíos.
- Crear guías.
- Rastrear paquetes.

## Tecnologías

- Python 3
- FastAPI
- Uvicorn
- UV (para gestión de entorno y paquetes)
